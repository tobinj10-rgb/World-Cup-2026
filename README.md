# Traction Live

A clean World Cup 2026 knockout tracker.

## Local setup

npm install
npm run dev

## Update match data

Edit `data/matches.json`.

## Deploy quickly

1. Push this folder to GitHub.
2. Go to https://vercel.com/new
3. Import the GitHub repo.
4. Click Deploy.
5. Share the live URL.

Official results only. Do not add predictions as scores.
